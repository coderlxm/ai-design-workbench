# AI设计工作台 第一阶段 MVP 核心流程文档

## 一、 第一阶段（MVP）核心目标与范围
* **核心目标**：完成从上传场景图到最终生成图的闭环。
* **业务范围**：
  * 场景图上传
  * 线稿生成
  * 场景 prompt 逆向
  * 模特图上传
  * 模特 prompt 逆向
  * prompt 自动组合
  * 最终生图
  * 生成记录保存

---

## 二、 核心业务流程与功能规范

### 功能一：选择模板图片 / 上传人物场景图
* **核心说明**：用户可以从系统模板库选择一张场景图，也可以上传自己的人物场景图。该图作为后续生成线稿和逆向场景 prompt 的基础。
* **业务输入项**：
  * 选择模板或上传图片（支持 JPG、PNG、WEBP）
  * 产品主题，如“母婴暖奶器广告图”
  * 场景设定（本期固定美国）
  * 输出比例（1:1 / 4:5 / 9:16 / 16:9）
* **交互规则**：
  * 上传后自动生成缩略图；用户可裁剪图片主体区域；
  * 系统自动检测图片是否有人物、是否清晰、是否符合生成条件；
  * 上传后展示预览图。

### 功能二：AI生成极简线稿图
* **核心说明**：系统将用户选择的模板图/场景图转成黑白极简线稿图，用于锁定最终生图的主体结构、人物位置、产品位置和构图关系。
* **固定 prompt**：
```text
转成极简线稿，细致线，清晰的结构，白底黑线
```
* **建议增强 prompt**：
```text
请将输入图片转换为极简黑白线稿图。要求：
1. 保留人物姿态、主体轮廓、产品位置、空间关系和主要构图；
2. 使用白色背景、黑色细线；
3. 线条清晰、干净、结构明确；
4. 不要生成阴影、颜色、文字、水印、复杂纹理；
5. 不要改变人物动作和画面构图；
6. 输出适合作为后续图像生成参考的结构线稿图。
```
* **交互与验收**：
  * 输出图必须为白底黑线；不允许出现文字、水印、彩色元素；用户可点击“重新生成线稿”。

### 功能三：AI逆向生成场景提示词
* **核心说明**：系统分析用户上传的场景图，逆向生成详细中文 prompt，用于描述画面中的产品主题、人物场景、外观、颜色、灯光、构图和风格。
* **用户提供 prompt**：
```text
我想使用nano banana的模型进行生图，我需要让你帮我反推一下这张图片的提示词，并把人物场景设定为美国，要求按照产品主题的外观，颜色，灯光，构图，风格。请帮我尽可能详细写出他的关键词，不要出现文字，整合成一段中文
```
* **建议系统增强 prompt**：
```text
你是一名专业AI图像提示词工程师。请基于输入图片，反推出适合 Nano Banana / Gemini 图像生成模型使用的中文提示词。

要求：
1. 人物场景统一设定为美国；
2. 结合用户输入的产品主题；
3. 详细描述画面主体、人物动作、产品位置、环境、外观、颜色、灯光、构图、镜头语言、视觉风格、材质、景深、画质；
4. 不要要求生成任何文字、logo、字幕、水印；
5. 不要输出分点，整合成一段完整中文；
6. 不要添加图片中不存在的关键主体；
7. 如果原图信息不足，可以补充合理的商业广告视觉细节，但不得改变核心构图。
```
* **交互与验收**：
  * 输出必须为一段中文；必须包含产品主题；必须包含美国场景设定；不得要求生成文字、logo、字幕；用户可手动编辑该 prompt。

### 功能四：选择模特 / 上传模特六视图
* **核心说明**：用户可以从模特库中选择已有模特，也可以上传模特六视图。六视图用于增强人物一致性，降低最终生图中脸型、发型、身材、肤色、服装细节漂移的问题。
* **业务输入项**：
  * 模特参考图（正面、左侧面、右侧面为必填；背面、半身、全身为建议选填）
* **交互与验收**：
  * 缺失视图时给出质量提示，但不强制阻断；
  * 上传模特图后展示六视图，系统可识别并展示每张图的视角标签；
  * 用户可删除、替换、重新上传任一视图。

### 功能五：组合最终 prompt 并调用模型生成图片
* **核心说明**：系统将以下内容组合为最终入参：1 产品主题 prompt；2 场景逆向 prompt；3 线稿图作为构图参考；4 模特图作为人物参考；5 输出尺寸、风格、禁止项、质量参数。
* **Prompt 组合逻辑**：
  最终Prompt = 产品主题说明 + 场景逆向Prompt + 构图约束 + 线稿参考约束 + 模特一致性约束 + 禁止项 + 输出质量要求。
* **建议最终 prompt 模板**：
```text
请基于上传的线稿图作为构图结构参考，基于上传的模特图作为人物外观参考，生成一张符合以下要求的高质量商业广告视觉图。

产品主题：{{product_theme}}

场景设定：{{scene_reverse_prompt}}

构图要求：
严格参考线稿图中的人物位置、姿态、主体轮廓、产品位置、空间关系和画面比例，不要大幅改变构图。

人物一致性要求：
参考上传的模特图，保持人物脸型、五官、发型、肤色、身材比例和整体气质一致，不要生成完全不同的人。

产品与风格要求：
画面需要突出产品主题，符合美国商业广告视觉风格，画面干净、高级、真实，灯光柔和，构图清晰，材质真实，适合电商广告和社媒投放。

禁止项：
不要出现任何文字、logo、字幕、水印、乱码、错误手指、畸形肢体、模糊面部、过度磨皮、低清画质、夸张变形、无关物品。

输出要求：
高清商业摄影质感，真实自然，细节清晰，主体突出，画面比例为 {{output_ratio}}。
```
* **交互与验收**：
  * 最终生成图应符合线稿构图；人物应接近模特参考图；图片中不得出现文字、logo、水印；产品主题必须清晰表达；
  * 失败后可基于同一输入重新生成；用户可查看最终 prompt；
  * 支持用户预览、下载、重试、保存。

---

## 三、 生成记录保存
最终生成结束后，系统自动保存并归档以下记录：
1. 一张符合产品主题的 AI 商品视觉图；
2. 一份可复用的完整组合 prompt；
3. 一套生成过程记录，包括线稿图、场景 prompt、最终模型入参；
4. 可二次编辑、重试、换模特、换场景、换风格的工作流资产。
