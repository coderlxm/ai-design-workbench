# AI 设计工作台演示操作手册（普通用户）

## 1. 演示前准备
- 电脑已安装 Node.js 18+ 与 pnpm。
- 项目已拉取到本地。
- 网络可访问豆包接口（当前项目已内置开发用 API Key）。
- 准备 1 张场景图 + 3 张模特图（正面/左侧面/右侧面）。

## 2. 启动项目
在项目根目录执行：

```bash
pnpm install
pnpm dev
```

浏览器打开：`http://localhost:5173`

## 3. 标准演示流程（建议按顺序）
1. 顶部栏填写产品主题（例如“户外运动水壶”），选择画幅比例和模型。
2. 在“场景输入”步骤：
   - 上传场景图；
   - 可选：点击“主体裁剪”，框选主体后确认。
3. 进入“线稿提取”：点击生成线稿，等待豆包返回结果。
4. 进入“场景反推”：点击生成场景提示词，右侧可手动微调。
5. 进入“模特输入”：上传至少 3 张必需视图（正面、左侧面、右侧面）。
6. 进入“最终产出”：点击生成最终图片。
7. 生成完成后可执行：查看最终 Prompt、下载图片、保存记录。
8. 打开“记录页（/records）”展示历史生成结果与复用能力。

## 4. 现场讲解建议（3 分钟版本）
- 第 1 分钟：说明输入内容（场景图 + 模特多视图 + 产品主题）。
- 第 2 分钟：展示 AI 自动生成线稿与场景 Prompt。
- 第 3 分钟：展示最终出图、下载与记录回看，强调流程闭环。

## 5. 常见问题
- 线稿或最终图生成失败：先检查网络，再重试。
- 提示“视图不足”：补齐正面/左侧面/右侧面。
- 效果不理想：优先微调场景 Prompt 后再生成。

## 6. 演示完成后的复位建议
- 清空模特视图。
- 上传新的场景图与主题，重复流程演示下一组案例。
