AI Design Workbench 当前完整逻辑总览

更新时间：2026-05-21（按当前代码状态）

纯净基线：pure/1.md 隐形步骤：pure/2.md 单模特参考图模式

1. 外显业务流程（4步）

步骤1：场景输入

  • 可选择模板场景或上传场景图。
  • 可选裁剪，裁剪图仅用于展示与主流程输入。

步骤2：模特输入

  • 上传 1 张模特参考图（单图内包含六视图）。
  • 默认预填示例第一张，可清空后重传。

步骤3：更换人物 Prompt

  • 本地模板生成 replacePrompt（不调外部接口）。
  • 约束：保构图/姿态/产品/婴儿，仅替换人物外观。

步骤4：最终生图

  • 输入：场景图 + 模特参考图 + 最终提示词。
  • 最终提示词 = replacePrompt + （可选）filterPrompt。

2. 隐形滤镜逆向步骤（可开关）

  • 开关字段：workspace.enableFilterReverse，默认开启。
  • 执行位置：步骤4生图前自动执行。
  • 输入图：优先原始场景图（selectedSceneAsset），回退到当前预览场景图。
  • 输出：prompts.filterPrompt，并拼接到最终提示词。
  • 失败策略：记录 warning，清空 filterPrompt，自动降级继续生图（不阻塞）。

3. 模型分流逻辑

selectedModelProvider ├─ 豆包 -> generateFinalArtworkWithDoubao() ├─ gpt-image-2 -> generateFinalArtworkWithGptImage2() └─ nano banana pro -> generateFinalArtworkWithNanoBananaPro()

豆包

  • 接口：/api/doubao/images/generations
  • 输入图：场景图 + 模特图（转换为可读 URL/dataURL）

gpt-image-2

  • 接口：/api/gpt-image-2
  • 优先图生图：multipart image[] + prompt；无图兜底 JSON。

nano banana pro

  • 接口：/api/nano-banana-pro
  • 请求：chat-completions 风格 messages + image_url。
  • 本地图会转 dataURL(base64)；公网 URL 直传 URL。

4. API 链路总览

A. 滤镜逆向（scene-reverse）

1) POST /api/scene-reverse/files/upload 2) POST /api/scene-reverse/chat-messages 3) 提取 answer -> prompts.filterPrompt

B. 最终生图（模型分支）

豆包: POST /api/doubao/images/generations gpt-image-2: POST /api/gpt-image-2 nano banana pro: POST /api/nano-banana-pro
